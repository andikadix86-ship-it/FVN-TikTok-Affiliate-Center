# Roadmap

## Phase 1: MVP Manual/CSV Mode

- Manual product input.
- CSV import.
- Product scoring.
- Prompt engine with Template Mode.
- Content draft library.
- Manual posting checklist.
- Posted content tracking.
- Manual analytics.
- Daily action plan.

## Phase 2: TikTok OAuth Stable

- Harden Login Kit error handling.
- Improve OAuth diagnostics.
- Store account status safely.
- Add clearer production setup checks.

## Phase 3: TikTok Shop API If Approved

- Add official TikTok Shop API adapter.
- Fetch real product data only through approved API access.
- Mark official API products as `REAL_API`.
- Keep demo/manual/CSV labels honest.

## Phase 4: Auto Posting If Scope Approved

- Evaluate official TikTok posting API access.
- Add auto-posting only if API scope is approved.
- Keep manual publishing as fallback.
- Never use unofficial posting or scraping.

## Phase 5: Multi-Account

- Support multiple TikTok accounts.
- Let users assign posted content and campaigns to specific accounts.
- Add account-level status and manual performance views.

## Phase 6: Shopee/Tokopedia Later

- Add marketplace-specific modules only after TikTok Affiliate MVP is stable.
- Keep non-TikTok marketplace logic separate from TikTok-specific modules.

## Phase 7: Separate Clipper App

- Keep AI Clipper as a separate application.
- Do not merge Clipper features into Affiliate Center.
- Share concepts only when useful, not folders or runtime coupling.

## Phase 8: Separate Creative Content Creator App

- Build a future creative content creator as its own standalone app.
- Keep advanced creative workflows separate from affiliate MVP operations.
