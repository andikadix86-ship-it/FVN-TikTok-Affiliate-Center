# MVP Testing Checklist

Use this checklist before sharing, deploying, or continuing development of FVN TikTok Affiliate Center.

## A. Product Flow

- [ ] Add manual product.
- [ ] Import CSV.
- [ ] Product score appears.
- [ ] Source badge correct.
- [ ] Demo data warning correct.

## B. Content Flow

- [ ] Generate content pack.
- [ ] Save draft.
- [ ] Copy hook, script, caption, hashtag, and CTA.
- [ ] Edit draft.
- [ ] Archive draft.

## C. Posting Flow

- [ ] Checklist posting works.
- [ ] Mark as ready.
- [ ] Mark as posted.
- [ ] Save TikTok video URL.
- [ ] Input performance manually.

## D. Campaign Flow

- [ ] Create 7-day campaign.
- [ ] Create 14-day campaign.
- [ ] Add performance data.
- [ ] Poor performance suggestion appears.

## E. Analytics Flow

- [ ] Total views, clicks, orders, and revenue correct.
- [ ] CTR correct.
- [ ] Conversion rate correct.
- [ ] Engagement rate correct.
- [ ] Best product, content, and campaign appears.

## F. Action Plan Flow

- [ ] Daily action plan appears.
- [ ] High-priority action appears.
- [ ] Demo-only warning appears if no real user product.

## G. TikTok OAuth Flow

- [ ] OAuth test page loads.
- [ ] Env status visible.
- [ ] Redirect URI validation works.
- [ ] Connect TikTok button works or shows clear error.

## H. Settings Flow

- [ ] Database status visible.
- [ ] AI provider status visible.
- [ ] TikTok OAuth status visible.
- [ ] TikTok Shop API status visible.
- [ ] Clear demo data only deletes DEMO.
